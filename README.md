## ⚙️ Installation

### Prérequis
- Node.js 18+

### Lancer en local
```bash
# Cloner le repository
git clone https://github.com/mhdgri/portfolio.git

# Installer les dépendances
npm install

# Lancer le serveur de développement
npm run dev

# Build pour la production
npm run build

# Prévisualiser le build
npm run preview
```
